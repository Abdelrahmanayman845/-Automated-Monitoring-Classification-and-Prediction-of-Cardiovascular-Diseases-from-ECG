1.Running the Interface

To run the interface, you must execute all the cells in the interface.ipynb file.
Ensure all the required libraries are installed.

2.Datasets

The project includes two datasets: MIT-BIH DATA and PHYSIO DATA.
Each dataset has its own folder.

3.Model Notebooks

Each folder contains notebooks for all the models used with the respective dataset.
Each model implementation is contained in a notebook named with the model name.
